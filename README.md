# Genpub

## What is it?

Publish a file to a given channel

Currently supports

  - google drive
  - twitter

## Usage

    genpub drive some_file.png

## Install

Install using either (as `sudo`)

  `./setup.py install`

Or

  `./setup.py develop`

The latter is most convenient if you will be editing the code.
